# MRI-Denoising-ResRicianNet
# Dataset
Dataset is not included due to size limitations. This project uses the Brain MRI Scans for Noise Removal Tasks available on Kaggle. 

🔗 Download Dataset: https://www.kaggle.com/datasets/shreyag1103/brain-mri-scans-for-noise-removal-tasks

After downloading, extract and place it in the following structure:

dataset/ └── DB/ ├── In/ └── GT/ ⚠️ Make sure the folder names match exactly, otherwise the code may not run.
